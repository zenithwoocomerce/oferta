Como funciona?

Pagamentos via PIX e Cartão de Crédito

O plugin [ZenithPay + Woocomerce] permite que você ofereça opções de pagamento via PIX e cartão de crédito nativamente para seus lojistas no WooCommerce. Com apenas alguns passos o lojista pode configurar os pagamentos!

O sistema calcula automaticamente os juros, processa os pagamentos e atualiza os pedidos (Após o cadastro do webhook).

É só instalar e usar!

A integração da Gateway ZenithPay é simples e rápida. Após instalar o plugin em uma loja Woocomerce com o checkout para atuar no Brasil, basta acessar as configurações de pagamento do WooCommerce, ativar a forma de pagamento desejada, inserir a chave de API fornecida na conta Zenith e configurar as preferências de pagamento.

Benefícios:

  • Integração Fácil: A integração da Gateway ZenithPay no WordPress é rápida e direta, basta instalar o plugin e configurar sem complicações.
  • Configuração Simples: Basta ativar a forma de pagamento desejada, inserir a chave Secret da API fornecida na conta ZENITH e configurar as preferências de pagamento em poucos passos.
  • Automatização de Pedidos: O plugin vem com um endpoint pronto que permite a atualização automática dos pedidos no WooCommerce, caso o pedido seja aprovado ou reembolsado.

Como Funciona:

  1. Instalação do Plugin: O lojista acessa os plugins wordpress, clica em [Adicionar Plugin] e depois em [Enviar Plugin], então basta clicar em [Escolher ficheiro] e selecionar o arquivo [zenithpay.zip], clicar em [Instalar agora] e por fim depois que a pagina recarregar basta descer clicar em [Ativar plugin]
  2. Configuração de Pagamento: Após a instalação basta acessar as configurações de pagamento do WooCommerce e clicar em Gerenciar na forma de pagamento desejada, então basta [inserir a chave Secret de API fornecida na conta Zenith] clicar na caixa [Ativar Zenithpay] e depois em [Salvar Alterações].
  3. Preferências de Pagamento: Nas configurações é possível personalizar as preferencias do pagamento conforme for necessário, como descontos em um método de pagamento específico ou o máximo de parcelas que o lojista quer disponibilizar na hora da compra.
  4. Automatização de Pedidos: Quando [ativado] o plugin cria o endpoint para zenithpay em:
    [https://siteexemplo.com/wp-json/zenithpay/v1/processar-pedido/].
    Basta criar um webhook na zenithpay para o endpoint após a ativação do plugin e já estará pronto para uso!

Com esta funcionalidade, os lojistas podem facilmente integrar e configurar o plugin [ZenithPay + Woocomerce], proporcionando uma experiência de pagamento eficiente e automatizada para seus clientes.

Tela para pagamentos via PIX

Após a instalação do plugin, a integração com o PIX é feita automaticamente na página de obrigado. No momento em que o cliente finaliza a compra utilizando o método PIX, ele é imediatamente redirecionado para uma tela de pagamento com o QR code PIX. Resumidamente, depois da instalação e configuração, o plugin já está pronto para o uso do lojista.

Benefícios:

  • Integração Automática: A integração com o PIX é configurada automaticamente após a instalação, facilitando o processo para o lojista.
  • Redirecionamento Imediato: Após a finalização da compra, o cliente é redirecionado automaticamente para uma tela com o QR code PIX, simplificando o pagamento.
  • Pronto para Uso: Após a instalação e configuração inicial, o plugin está pronto para uso, permitindo que os lojistas aceitem pagamentos via PIX sem complicações adicionais.

Como Funciona:

  1. Configuração Automática: Após a configuração do método de pagamento via PIX, a tela de pagamento estará pronta para uso.
  2. Redirecionamento para Pagamento: Após a finalização da compra com o método PIX, o cliente é redirecionado para uma tela com o QR code PIX para concluir o pagamento.

Configuração de Desconto Personalizado para cada Método de Pagamento

Através do gerenciamento de cada método de pagamento, o lojista pode escolher oferecer uma porcentagem de desconto no pagamento escolhido. Ou simplesmente deixar 0 para não oferecer descontos.

Benefícios:

  • Flexibilidade Promocional: Permite que os lojistas ajustem dinamicamente as porcentagens de desconto, facilitando a criação de promoções sazonais, campanhas de marketing específicas ou incentivos para métodos de pagamento preferenciais.
  • Aumento nas Vendas: Ao oferecer descontos atrativos, os lojistas podem incentivar os clientes a optarem por métodos de pagamento específicos, potencialmente aumentando a conversão e as vendas gerais.
  • Melhoria na Experiência do Cliente: Oferecer descontos claramente aplicados no checkout melhora a experiência de compra, aumentando a satisfação do cliente e reduzindo a taxa de abandono do carrinho.

Como Funciona:

  1. Configuração Simples: Nas configurações do WooCommerce, os lojistas podem definir a porcentagem de desconto desejada individualmente para cada método de pagamento disponível, como Cartão de Crédito ou PIX.
  2. Aplicação Automática: O desconto configurado é automaticamente aplicado ao subtotal dos produtos durante o processo de checkout, sendo claramente exibido ao cliente.
  3. Indicador de Desconto: Um indicador visual no checkout mostra ao cliente a porcentagem exata do desconto aplicado, destacando o benefício de usar o método de pagamento selecionado.

Tratamento de erros

Durante o processo de pagamento, o sistema valida os dados do cliente e caso algum erro ocorra no processo, é imediatamente mostrado para o cliente.
